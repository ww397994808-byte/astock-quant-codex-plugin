# 阶段推进裁判

## 目标

解决 AI 无限研究的问题。每轮研究、回测、审计、模拟盘或 QMT 检查结束后，必须明确判断：继续研究、停止重写、进入下一阶段，还是被阻断。

## 两种模式

严格教学模式：
- 默认模式。
- 达到推进条件时，必须建议进入下一阶段。
- 未达到门槛时，只允许围绕阻断项修复。
- 不允许无目标调参或继续研究。

研究员模式：
- 用户明确要求后才启用。
- 可以继续研究，但必须限定目的。
- 可以做鲁棒性测试、样本外验证、压力测试、策略范式对照、补数据。
- 不能绕过审计、模拟盘、QMT 和人工确认。

## 推进判断中文词

- 建议进入下一阶段：证据足够，应停止无意义研究。
- 建议进入下一阶段，但允许继续研究：研究员模式下已满足推进条件，继续研究需有明确目的。
- 仅允许围绕阻断项继续研究：存在明确问题，只能修这个问题。
- 停止当前方向，重新定义策略：方向错误、证据不足、过拟合严重或数据不支持。
- 当前被阻断，不能推进：数据、环境、QMT、审计或风控条件不足。

## 反无限研究规则

在严格教学模式下，只要满足以下条件，必须建议进入下一阶段：

- 策略想法完整，研究计划固定，进入回测。
- 回测审计通过，进入模拟盘。
- 模拟盘观察达标，进入 QMT 只读。
- QMT 只读通过，进入实盘前检查。
- 实盘前检查通过，进入真实运行人工确认。

不要继续建议“再优化一下”“再研究一下”“再试几个参数”，除非报告显示有明确阻断项。

## 输出模板

```text
当前模式：严格教学模式
当前阶段：回测已通过
推进判断：建议进入下一阶段

为什么：
审计已通过，回测假设没有漂移，没有发现高风险问题。

下一步：
进入模拟盘观察。

下一条命令：
python cli.py paper ...

不要做：
不要继续随机调参，也不要跳过模拟盘直接讨论实盘。
```

## 研究员模式继续研究模板

```text
当前模式：研究员模式
当前阶段：回测已通过
推进判断：建议进入下一阶段，但允许继续研究

为什么：
当前证据已经满足进入模拟盘的最低条件。继续研究可能增加过拟合风险。

允许继续研究的范围：
鲁棒性测试、样本外验证、手续费压力测试、策略范式对照。

不要做：
不要随机搜索参数，只挑收益最高的一组。
```
