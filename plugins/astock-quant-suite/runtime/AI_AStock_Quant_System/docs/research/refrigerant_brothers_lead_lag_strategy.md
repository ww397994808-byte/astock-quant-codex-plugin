# 制冷剂三兄弟相关性策略草案

## 研究目的

验证巨化股份、三美股份、永和股份三只制冷剂相关股票之间是否存在可交易的领涨-补涨关系。

本策略不是假设“三只股票一定同涨”，而是检验：

- 当其中一只股票率先强势上涨时，另外两只股票是否会在之后 1 到 5 个交易日内补涨。
- 这种补涨是否扣除交易成本、滑点、T+1 约束后仍有稳定收益。
- 领涨信号是否只在某些市场环境下有效，例如制冷剂价格上涨、化工板块走强或市场风险偏好较高。

## 固定股票池

| 名称 | 代码 | 角色 |
| --- | --- | --- |
| 巨化股份 | 600160.SH | 制冷剂龙头/领涨候选 |
| 三美股份 | 603379.SH | 制冷剂弹性标的/补涨候选 |
| 永和股份 | 605020.SH | 制冷剂弹性标的/补涨候选 |

## 策略类型

- 资产版本：A股
- 周期：日线
- 范式：固定三股票池轮动 / 领涨-补涨 / 相关性策略
- 价格处理：point_in_time_qfq
- 执行假设：收盘后产生信号，下一交易日开盘或 VWAP 附近执行
- 禁止假设：不能用当日收盘信号在当日收盘价成交

## 核心信号

每日收盘后，对三只股票分别计算：

- 当日收益率：`ret_1d = close / close.shift(1) - 1`
- 3 日收益率：`ret_3d = close / close.shift(3) - 1`
- 20 日相对强度：`rs_20d = close / close.shift(20) - 1`
- 成交量放大：`volume_ratio = volume / volume.rolling(20).mean()`
- 跟涨程度：候选股票当日涨幅是否明显低于领涨股票

## 领涨定义

一只股票在收盘后满足以下条件，视为当日领涨股：

- `ret_1d >= 3%`
- `volume_ratio >= 1.5`
- `ret_1d` 在三只股票中排名第 1
- 与股票池第二名的当日涨幅差距 `>= 1.5%`

如果当天有涨停、停牌、异常数据，需要单独标记，不能直接视为普通可交易信号。

## 买入规则

收盘后找到领涨股后，在另外两只股票里寻找补涨候选：

- 候选股票当日涨幅低于领涨股至少 `1.5%`
- 候选股票当日涨幅不超过 `4%`，避免已经大幅跟涨后追高
- 候选股票过去 20 日没有明显破位，例如收盘价不低于 20 日均线的 `95%`
- 候选股票次日不是停牌、ST、涨停无法买入状态

买入方式：

- 若两个候选都满足条件，等权买入两个候选
- 若只有一个满足条件，只买入该候选
- 若三个股票同涨，没有明显落后者，则不交易

## 卖出规则

持仓后满足任一条件退出：

- 持有满 3 到 5 个交易日
- 持仓收益达到 `+5%`
- 持仓收益低于 `-3%`
- 原领涨逻辑失效：候选股票继续弱于股票池平均，且化工/制冷剂相关板块同步转弱
- 出现停牌、ST、重大异常数据时进入风控处理

## 仓位与风控

- 单次策略总仓位不超过账户权益的 `30%`
- 单只股票不超过账户权益的 `15%`
- 同一时间最多持有两只补涨候选
- 连续 3 次亏损后暂停策略 5 个交易日
- 单笔最大亏损按 `3%` 止损估算，单次策略风险不超过账户权益的 `1%`

## 研究检验

第一阶段先做统计检验，不直接上策略回测：

1. 计算三只股票日收益率相关矩阵。
2. 计算滚动 60 日相关性，观察相关性是否稳定。
3. 做 lead-lag 检验：A 股票上涨后，B/C 在未来 1、3、5 日的平均收益和胜率。
4. 区分领涨来源：巨化领涨、三美领涨、永和领涨分别统计。
5. 比较“买补涨者”和“直接买领涨者”的表现，确认补涨逻辑是否真的优于追领涨。

第二阶段再做策略回测：

- 入样本：参数选择与逻辑验证
- 出样本：只验证，不再调参
- 指标：年化收益、最大回撤、胜率、盈亏比、交易次数、单票贡献、分年份表现
- 稳健性：阈值从 2% 到 5%、持有期从 2 到 7 日、成交量倍数从 1.2 到 2.0 分组测试

## 失败条件

出现以下任一情况，策略应判定为研究失败或需要重写：

- 未来 1 到 5 日补涨收益不显著，胜率接近随机。
- 收益主要来自少数几次行情，分年份不稳定。
- 巨化、三美、永和之间相关性只在同涨同跌时存在，没有稳定的滞后关系。
- 买补涨者不如直接买领涨者，或者不如等权持有三只股票。
- 扣除交易成本和无法成交场景后收益消失。

## 下一步命令建议

当前本地预检器未能自动识别这个三股票固定池，因此下一步应先补齐数据并做专门的 lead-lag 统计脚本，而不是直接套单标的回测模板。

建议下一步：

```bash
python3 cli.py update-data --symbol 600160.SH --timeframe 1d --adjust point_in_time_qfq
python3 cli.py update-data --symbol 603379.SH --timeframe 1d --adjust point_in_time_qfq
python3 cli.py update-data --symbol 605020.SH --timeframe 1d --adjust point_in_time_qfq
```

拿到数据后，再建立专门的三股票 lead-lag 研究脚本，输出：

- 相关性矩阵
- 领涨事件表
- 补涨收益统计
- 参数敏感性结果
- 是否进入正式回测的结论
