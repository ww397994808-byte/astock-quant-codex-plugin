# Fixed Templates

These templates are copied from the bundled runtime and are kept as stable reference material for course delivery and strategy development.

- `config/qmt_config.example.yaml`: safe QMT config shape; real local config is never committed.
- `backtest_templates/`: timing, swing, grid, rotation, stock-selection, portfolio, pair-trading, and event-driven templates.
- `strategies/`: baseline strategy implementations and registry patterns.

Use the runtime CLI to create evidence. Do not treat a copied template as validated strategy performance.
